Справочный материал о программе.
Данный репозиторий представляет собой площадку для разработки программного продукта, предназначенного для ассистирования математикам-информатикам. Программа обеспечивает перевод целых и дробных чисел в различные системы счисления, определяемые самостоятельно пользователем. Операции перевода чисел производятся автоматически, с контролем вводимых данных, с целью обеспечения корректности ответов на запросы пользователя. В случае недостаточного опыта пользователя в использовании данного программного продукта, реализована функция, которая создает образцовый запрос, который возможно выполнить с помощью программы.
